# MORA - Next Generation Social Platform

## Design Philosophy
MORA is designed to be the most premium, fluid, and emotionally intelligent social platform. 

### Key Features
- **Deep Space Theme**: A custom dark mode palette using `hsl(240 10% 3.9%)` as the base.
- **Electric Gradients**: Vibrant teal to purple gradients (`text-gradient`) that pop against the dark background.
- **Glassmorphism 2.0**: Refined glass effects with `backdrop-filter` and subtle white borders.
- **Micro-interactions**: Framer Motion powers every interaction from button clicks to input validation.
- **Responsive**: A "mobile-first" but "desktop-class" layouts.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   
3. **Open in Browser**
   Visit `http://localhost:5173` (or the port shown in terminal)

## Project Structure
- `src/index.css`: The heart of the design system (variables, utilities).
- `src/components/LoginCard.tsx`: The ultra-premium login component.
- `src/pages/LoginPage.tsx`: The main landing experience.
- `src/components/ui`: Reusable Shadcn-style components (Button, Input, Label).

Enjoy the experience.
